# SunProject
gin框架的简单封装

# 目录结构
```text
  ├─config           配置文件目录
  │  ├─database.go       数据库配置文件
  │  ├─log.go            日志配置文件
  │  ├─redis.gp          redis配置文件
  │  └─common.php        公共函数文件
  ├─models           数据库模型目录
  ├─routes             路由目录
  ├─middleware       中间件目录
  │  ├─RouterMiddleware.go  路由中间件

```
